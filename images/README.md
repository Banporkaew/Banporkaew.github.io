โฟลเดอร์นี้เก็บรูปภาพของเมนู

แนวทางการตั้งชื่อไฟล์ (ต้องตรงกับที่ระบบแม็ปไว้โดยอัตโนมัติ):
- ระบบจะใช้ชื่อเมนูเป็นฐาน และแปลงเป็น slug ดังนี้:
  1. ตัดช่องว่าง → แทนด้วย `-`
  2. ลบตัวอักษรพิเศษ/เครื่องหมายวรรคตอน (ยกเว้นตัวอักษรไทยและตัวเลข)
  3. ใช้ตัวพิมพ์เล็ก
- ตัวอย่าง:
  - "ลาบหมู" → `ลาบหมู.jpg`
  - "ข้าวผัดปู (จานใหญ่)" → `ข้าวผัดปู-จานใหญ่.jpg`

ใส่รูปจริงลงในโฟลเดอร์นี้ตามชื่อไฟล์ที่แม็ปไว้ ตัวอย่างไฟล์ (สร้างโดยผู้ใช้):
- ลาบหมู.jpg
- ข้าวผัดปู-จานใหญ่.jpg

ถ้ารูปยังไม่มี ระบบจะแสดงรูป placeholder อัตโนมัติ

หากต้องการให้ผมสร้างรายการแม็ปชื่อ→ไฟล์ให้เป็นไฟล์ JSON ให้บอกมา ผมจะสร้างไฟล์ `images/mapping.json` ให้โดยตรง
